\# Bilan des Labs Git — Le Gardien



\## 3 choses apprises



1\. Git permet de sauvegarder l'historique de son travail avec git add et git commit.

&#x20;  Chaque commit est daté, signé et récupérable.



2\. Les branches permettent de travailler en parallèle sans gêner les autres membres

&#x20;  de l'équipe. git switch -c crée et bascule sur une nouvelle branche.



3\. Un conflit Git survient quand deux membres modifient la même ligne du même fichier.

&#x20;  Il se résout en lisant les marqueurs, en choisissant la bonne version,

&#x20;  et en faisant git add + git commit.
